# MHH Prototype Equipment

Adds overpowered Prototype equipment, armor, robots, and technologies to Factorio.

Contains:
- mhh-prototype-power-armor (dynamic grid, 90% resistances, flight with Space Age / SE)
- mhh-prototype-thruster-suit (SE-only craftable upgrade)
- Equipment: battery, energy shield, fusion reactor, exoskeleton, personal laser defense, personal roboport
- Construction and logistic robots, stationary roboport
- Recipes for all items (disabled by default, unlocked via technology)
- 10 individual technologies per equipment/armor/robot category

Settings:
- Halve equipment size: makes equipments half size to fit more into your armor
- Enable recipes: allows crafting Prototype equipment late-game
- Equipment strength: Balanced / Overpowered / Cheaty tiers

Rebranded from MHH_RR_Items. Requires only base Factorio 2.0.
