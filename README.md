# MHH Prototype Equipment

Adds overpowered Prototype equipment, armor, robots, and technologies to Factorio.

Contains:
- mhh-prototype-power-armor (11x12 grid, 90% resistances, infinite durability, flight with SE)
- Equipment: battery, energy shield, fusion reactor, exoskeleton, personal laser defense, personal roboport
- Construction and logistic robots
- Recipes for all items (disabled by default, unlocked via technology)
- 9 individual technologies per equipment/armor/robot category

Settings:
- Extra cheese: halves equipment sizes
- Enable recipes: allows crafting Prototype equipment late-game
- Progressive cost: how many previous-tier items required per craft

Rebranded from MHH_RR_Items. Requires only base Factorio 2.0.
